# AI Coding 潜力与“人与技术互相塑造”

## 潜力的激发

AI Coding 作为一种技术集群的潜力将如何被进一步激发？

有赖于更多的人投入到“采用 AI Coding Agent 工具 / 自动化编程与软件开发产品进行软件开发和产品管理”的实践之中，以及从事相关工作。

## 人与技术的互动

同步地，人的潜力也恰恰在“采用 AI Coding Agent 工具 / 自动化编程与软件开发产品进行软件开发和产品管理”的实践过程中，在他们与“AI Coding Agent 工具 / 自动化编程与软件开发产品”及其技术衍生物的互动过程中，在他们参与“AI Coding Agent 工具 / 自动化编程与软件开发产品”所构造或组成的媒介环境时，被激发与重构了。

这是一个人的主体性及其实践过程与技术媒介反复互动并且相互塑造的过程。

从“人类潜能与技术生态共演”的哲学与社会学层面，而不仅仅是“工具效率”的层面来看，所谓“AI Coding 的大势”不仅是一场技术迭代，更是一场“人与技术互为媒介、互相塑造”的生态演进。

## 共生飞轮

技术潜力的“得势”，依赖人的规模化与深度实践。
人的潜力与主体性的“得势”，在技术媒介中被激发与重构。

技术成为一面镜子，也是一把凿子，让人看见并塑造自己未曾察觉的潜力。

**AI编程的未来，靠什么来激活？**
答案很简单：靠我们更多的人，去动手用它。

当更多人开始真正使用AI编程工具来构建软件、管理产品时，这些工具的潜力才会被彻底激发。它会在我们解决实际问题的过程中，变得越来越聪明、越来越好用。

更奇妙的是，这个过程也在重塑着我们自己：
- 原本觉得编程高不可攀的人，现在能借助AI把自己的想法变成产品原型。
- 开发者可以从重复代码中解放出来，去思考更核心的架构与创意。
- 产品经理能更快地验证创意，更专注于理解用户。

我们与工具，在互相塑造中共同成长。 我们用它，它就变得更强；它变强了，又反过来拓展了我们的能力边界，激发出我们自己都未曾察觉的潜能。

这是一个彼此点亮、共同进化的飞轮。
